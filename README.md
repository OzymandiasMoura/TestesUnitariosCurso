# Testes Unitários - Curso

Este repositório contém o projeto desenvolvido durante o curso de testes unitários disponível na Udemy.

O objetivo do projeto é demonstrar conceitos fundamentais de **testes unitários em aplicações Java Spring**, utilizando ferramentas modernas do ecossistema Java.

## 📚 Curso

Este projeto foi desenvolvido acompanhando o curso:

Testes unitários com JUnit 5

Link do curso:

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar e demonstrar:

- Criação de **testes unitários**
- Uso de **JUnit**
- Estruturação de testes automatizados
- Boas práticas para testes em aplicações Java
- Organização de código para facilitar testabilidade

## 🛠 Tecnologias Utilizadas

- Java
- JUnit
- Maven
- IntelliJ IDEA

## 📁 Estrutura do Projeto

O projeto segue a estrutura padrão de aplicações Java utilizando Maven:

- `main` contém o código da aplicação
- `test` contém os testes automatizados

## ▶️ Como executar os testes

Para executar os testes utilizando Maven:

```bash
mvn test
```
